# Lesson 22: Function Pointers and Callback Functions [:video_camera:](https://youtu.be/OIseV5lcx8w?si=KPT6udQ3wDeKk_GQ)

This lesson explores function pointers, demonstrating how to store function addresses in variables and pass them as arguments to other functions to implement callbacks.

## Function Pointers
- **`mainptr.c`** - Function pointer declaration, initialization, and usage, showing how a function's memory address can be manipulated like any other pointer.
- **`helloptr.c`** - Passing function pointers as parameters (callback functions).

## Callback Function
- **`sortptr.c`** - Implementation of a comparison function for `qsort()`, illustrating the use of callback functions in standard library functions.
